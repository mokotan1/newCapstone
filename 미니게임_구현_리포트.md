# newCapstone 미니게임 구현 리포트 (Prototype)

주인님, 요청하신 구글 슬라이드 기획안에 따라 `newCapstone` 프로젝트 내에 뱀파이어 서바이벌 스타일의 미니게임 프로토타입 스크립트를 작성했습니다.

## 🛠 구현된 주요 기능 (Scripts)

1.  **MiniGameManager.cs**: 전체 게임 루프 관리 (체력 5회, 적 스폰, 승리/패배 조건).
2.  **MiniGamePlayer.cs**: 8방향 이동 및 마우스 클릭 시 **십자가 휘두르기(Melee)** 공격 구현. (공격 시 플레이어 앞쪽에 히트박스 생성, 쿨타임 0.5초 적용)
3.  **MiniGameEnemy.cs**: 플레이어를 추적하며 에너지 탄환 또는 휘두르기 공격(`Melee` 태그)에 맞으면 소멸하는 기본 AI.
4.  **MiniGameExit.cs**: 맵 우측 끝(4500, 1500) 구역 도달 시 탈출 트리거.

## 📐 설정값 (기획안 반영)
- **맵 사이즈**: 5000 x 3000 px
- **플레이어 체력**: 5회 (적 충돌 시 차감 및 15px 넉백)
- **공격 방식**: 근접 휘두르기 (Melee Hitbox 사용)
- **탈출 조건**: 우측 끝 출구 구역 도달
- **사망 조건**: 체력 0 도달 시 마지막 세이브 시점(벽돌방)으로 복귀 로직 준비

## 🚀 유니티 적용 방법
1.  `Assets/Scenes` 폴더에 `MiniGame` 씬을 새로 생성합니다.
2.  빈 GameObject를 만들고 `MiniGameManager` 스크립트를 붙입니다.
3.  플레이어, 적, 투사체(에너지), 출구용 스프라이트를 각각 만들고 해당 스크립트들을 연결합니다.
4.  각 오브젝트에 `BoxCollider2D`와 `Rigidbody2D`를 설정하고 Tag(`Player`, `Enemy`, `Projectile`)를 지정하십시오.

이미지는 넣지 않았으며, 유니티 에디터에서 기본 프리미티브(Square, Circle 등)로 바로 테스트 가능한 상태입니다. 추가 명령 있으시면 바로 처리하겠습니다!
